A cerca del proyecto

Este website se ha desarrollado como Trabajo Fin de Grado del Curso de Complementos de Formacion de Adaptacion a Grado en Ingenieria de Sistemas de Información.

Decision Manager permite al usuario almacenar información contenida en ficheros de tipo Excel y definir validaciones, a través de un formulario Web, basados en las columnas del Excel procesado pudiendo obtener el resultado de los registros que cumplan esas validaciones

Decision-Manager está desarrollado por David Gómez Castellanos

Esta Web ha sido desarrollada utilizando

Java Hibernate Maven Camunda DMN Spring Thymeleaf Bootstrap
